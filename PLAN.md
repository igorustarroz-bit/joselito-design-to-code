- **Icons**: `src/components/Icon/` (página "🏷️ Brand Assets" de Figma, 132 iconos únicos + componente "Icon Sizer", COMPONENT_SET node `49026:6521`, 5 variantes de tamaño). Primer uso real del pipeline SVGR (`vite-plugin-svgr`, declarado desde el scaffolding inicial pero sin usar hasta ahora) — con 132 iconos ya no compensa recrearlos a mano en `icons.jsx` como se hizo en Button/NavButton/Accordion/Title/CardProduct (1-2 iconos cada uno). Los 132 archivos se guardan en `src/assets/icons/{Nombre}.svg` (convención `PascalCase`) y `Icon.jsx` los carga todos con `import.meta.glob('../../assets/icons/*.svg', { eager: true, query: '?react', import: 'default' })`, construyendo un mapa `nombre → Componente` en build time; la prop `name` selecciona cuál renderizar. **Limitación de entorno reutilizada de sesiones anteriores**: `figma.com` está bloqueado por el proxy de red del sandbox (`403 blocked-by-allowlist`), así que no se pudo usar `download_assets` + descarga HTTP como indicaba el plan original — se exportó cada SVG con `use_figma`/`node.exportAsync({format:'SVG'})` + `figma.base64Encode`, devuelto inline en la respuesta del tool (sin red), y decodificado con Python. Cada SVG se guardó con `stroke="black"`/`fill="black"` sustituidos por `currentColor` (dejando `fill="none"` intacto) y sin los `width="32" height="32"` fijos del root `<svg>` (se conserva el `viewBox`), para que el tamaño lo controle `icon.css` vía la prop `size`. Tamaños (`xxs`/`xs`/`s`/`m`/`l`) reutilizan los primitivos ya existentes `--spacing-fx-3/4/5/6/8` (12/16/20/24/32px, numéricamente idénticos a los 5 valores de "Icon Sizer" en Figma) — no se crearon tokens de tamaño nuevos. **Duplicados detectados en Figma**: 3 iconos (`MapPin`, `Quotes`, `House`) estaban insertados dos veces en la página con el mismo nombre — la segunda instancia visualmente idéntica salvo un `stroke-linecap` añadido; se conservó solo el nodo canónico (ID más bajo) de cada par, documentado en `Icon.mdx`. Accesibilidad: con `title` se marca como imagen (`role="img"` + `aria-label`, en vez de inyectar un `<title>` interno vía SVGR por simplicidad); sin `title`, decorativo (`aria-hidden`/`focusable="false"`). Icono no encontrado por `name` → `null` + `console.warn` en dev, sin romper la UI en producción. Documentación completa en `Icon.mdx`, incluyendo la galería completa de los 132 iconos (`AllIcons`, grid propio en vez de `VariantSwatch` por legibilidad con tantos elementos).
