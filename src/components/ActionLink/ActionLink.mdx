# Action Link

## Intro

`ActionLink` es un enlace de acción subrayado, sin fondo: "ver más", "leer artículo", enlaces dentro de tarjetas o listados. Corresponde al componente **UI04-Button-Action-Link** de Figma (`node 49038:9486`).

## Anatomía

Una fila horizontal centrada con, de izquierda a derecha:

1. **Icono izquierdo** (opcional) — 24px (L), 16px (M/S)
2. **Texto** — mayúsculas, tipografía CTA, subrayado (`border-bottom`)
3. **Icono derecho** (opcional) — mismo tamaño que el icono izquierdo

A diferencia de `Button`/`ButtonIcon`, no hay caja de fondo: el color de "fondo/borde" de Figma se traduce como el color del subrayado.

## Temas

Cambia el selector **"Tema"** en la barra de herramientas de Storybook para ver `ActionLink` en los 5 temas semánticos — vía los tokens `--color-btn-link-primary-*`.

## Comportamiento

- **Hover** y **Focus** son automáticos (`:hover`, `:focus-visible`) — no se controlan por prop.
- **Selected** añade `aria-current="page"`. Figma no define una variante "Selected" para este componente, pero el token existe en Semantic-Color y se aplica por consistencia con el resto de botones del sistema.
- **Disabled** desactiva la interacción (`pointer-events: none`) y aplica los tokens de estado deshabilitado.
- Si se pasa `href`, se renderiza como `<a>`; si no, como `<button type="button">`.

## Propiedades

| Nombre | Descripción | Por defecto | Control |
|---|---|---|---|
| `text` | Texto del enlace | `"Button"` | text |
| `size` | Tamaño: `l`, `m`, `s` | `"l"` | select |
| `showIconLeft` | Muestra el icono izquierdo | `false` | boolean |
| `showIconRight` | Muestra el icono derecho | `false` | boolean |
| `selected` | Marca el enlace como activo (`aria-current="page"`) | `false` | boolean |
| `disabled` | Desactiva la interacción | `false` | boolean |
| `href` | Si se indica, renderiza un `<a>` en vez de `<button>` | — | text |

## Accesibilidad

- El estado `selected` se comunica con `aria-current="page"`.
- El estado `disabled` usa el atributo nativo `disabled` en `<button>`, o `aria-disabled` + `pointer-events: none` cuando se renderiza como `<a>`.
- El foco visible (`:focus-visible`) usa un token de color con contraste suficiente en los 5 temas.
- Los iconos son puramente decorativos; el texto visible ya describe la acción.
