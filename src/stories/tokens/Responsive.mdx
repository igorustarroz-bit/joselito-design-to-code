# Responsive

## Intro

Los tokens **Responsive** son los valores que cambian según el ancho de viewport: 7 breakpoints (modos en Figma), un grid de 12 columnas, la escala tipográfica fluida, espaciados y radios de esquina. Se generan desde la colección `Responsive` de variables de Figma.

Se implementan como variables CSS definidas en `:root` y sobrescritas mobile-first dentro de `@media (min-width: ...)` — así cualquier componente que use `var(--text-title-3)`, `var(--spacing-r-8)`, etc. es responsive automáticamente, sin tener que escribir clases distintas por breakpoint.

Esta página documenta solo los tokens **fundacionales**. Los valores responsive específicos de un componente (tamaños de `Button`, `Forms` o la tipografía de `Input`) se documentan en la página de ese componente cuando se construya, no aquí.
